# intro-to-ai-coding-agents

Este repositorio contiene los materiales para el taller de "Introducción a AI Coding Agents".

slides/ tiene las diapositivas del taller.
data/ contiene los conjuntos de datos utilizados en los ejercicios del taller.
AGENTS.md contiene la guía de codificación para los AI Agents.
.devcontainer/ contiene la configuración del entorno de desarrollo para Visual Studio Code.
